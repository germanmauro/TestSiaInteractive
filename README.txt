Una vez clonado el repositorio:
1- Abrir una terminal en el directorio server.
2- Ejecturar npm index.js para correr el servidor.
3- Abrir al menos en dos ventanas del navegador el archivo index.html.
4- Verificar en la consola de cada ventana abierta que se indique que la conexión está establecida
5- Probar clickeando play/pause en cualquiera de los videos en las ventanas del navegador
